# Gomoku-Full-Stack-Game

## How to Play
    1. Move current directory to gomoku-service
    2. start the server with - npm start
    3. Move current directory to gomoku-client
    4. start the client with - npm start
    5. Login/Register to the game
    6. Start Playing
